# health-bmi-calculator-
The logic for a BMI calculator involves dividing a person's weight in kilograms by the square of their height in meters, according to the formula: BMI = weight (\(kg\)) / (height (\(m\))\({}^{2}\)). To use it, first, ensure weight is in kilograms and height is in meters (if your height is in cm , divide by 100 to convert).
